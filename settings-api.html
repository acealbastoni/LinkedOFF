🔑 إعدادات الـ API
إدارة روابط وبيانات الاتصال لكل خدمة خارجية يستخدمها التطبيق. التعديلات تُحفظ محلياً على جهازك وتُطبَّق فوراً على الطلبات القادمة.
ℹ️ الروابط الافتراضية تعمل بشكل طبيعي ولا تحتاج تعديلاً. عدّل فقط إذا أردت تحويل خدمة معينة إلى نسخة مختلفة من الـ API (مثل بيئة اختبار أو deployment جديد). الضغط على ↩ استعادة الافتراضي يُلغي أي تعديل ويعود للرابط الأصلي.
🔐
خدمة المصادقة
تسجيل الدخول، إنشاء الحسابات، إدارة الجلسات، جلب المستخدمين وتحديث بياناتهم
—
login register changePassword listUsers deleteUser updateProfile
رابط Google Apps Script المنشور لخدمة المصادقة. يجب أن ينتهي بـ /exec ولا يحتوي على أي query string.

خدمة الوظائف
جلب قوائم الوظائف، البحث، الفلترة، والتصفح بالصفحات من Google Sheets
—
getJobs searchJobs getJobById getCategories
رابط Google Apps Script المنشور لخدمة الوظائف.
مفتاح API المُرسَل مع كل طلب في حقل key.
